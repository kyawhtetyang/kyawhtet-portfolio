Flask Login User - First Complete CRUD Project
Project Diary

This was my first complete end-to-end project. It taught me the “junior‑dev core” of web development: CRUD, authentication, and clean project architecture. I still remember how exciting it felt to see a full app working together.

I built a Flask app with login/logout, role-based access (admin/editor/viewer), and full CRUD for users. I learned password hashing, session handling, decorators, and how to structure a Flask project so routes, auth, and data logic stay clean.

On the frontend, I used Jinja templates with a shared base layout. That helped me understand how backend routes, templates, and user auth connect in a real app.

Skills gained: Flask fundamentals, CRUD patterns, authentication + sessions, password hashing, role-based access, template inheritance, and REST-style API endpoints.
