FastAPI CRUD - Moving Beyond Flask
Project Diary

I learned FastAPI later than I wanted. I already knew Flask well, but I realized the industry is shifting toward FastAPI for modern APIs, typing, and performance. This project was my way to catch up and move forward.

I built a full CRUD app with FastAPI, server‑rendered HTML templates, and a SQLite database. It includes user authentication (register/login) and a clean, minimal UI with no JavaScript.

It helped me see how FastAPI encourages clearer structure and validation, while still letting me reuse the web fundamentals I learned from Flask.

Skills gained: FastAPI fundamentals, CRUD patterns, auth flow with templates, SQLite integration, and clean API-first structure.
