House Price Regression - Supervised Learning Focus
Project Diary

After exploring other ML paths, I wanted to focus on supervised learning progress. This project is a clean, end-to-end regression pipeline for predicting house prices using structured features.

I worked through the full workflow: data cleaning, missing-value handling, EDA, feature engineering, training, and evaluation. I compared a linear regression baseline with Ridge and Lasso (GridSearchCV), then selected the best model based on RMSE, MAE, and R².

The key lesson was how much data prep and feature choices matter for regression performance, and how to validate models without leaking information across train/test splits.

Skills gained: supervised regression pipeline design, feature engineering, model comparison (Linear/Ridge/Lasso), hyperparameter tuning, and evaluation with RMSE/MAE/R².
