# CloudLanguage MVP

CloudLanguage is a mobile-first language learning MVP designed around Burmese learners.

Recruiter highlights:

- React and TypeScript frontend
- FastAPI backend direction
- structured lessons, progress tracking, and review loops
- product-oriented build rather than just a static landing page

What it proves:

- full-stack product thinking
- ability to build user-facing learning workflows
- experience connecting frontend presentation with backend logic
