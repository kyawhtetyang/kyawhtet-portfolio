# Portfolio Site

The AI/ML Portfolio site is a recruiter-facing frontend built to present projects clearly rather than functioning as a generic gallery.

Recruiter highlights:

- React and TypeScript frontend
- Vite build workflow
- outcome-focused project summaries
- direct links to demos and GitHub repositories
- Ask section designed to evolve into an LLM-backed assistant

What it proves:

- product presentation judgment
- frontend implementation skill
- ability to package technical work for non-technical review
