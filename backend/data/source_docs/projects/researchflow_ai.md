# ResearchFlow AI

ResearchFlow AI is the strongest AI engineer proof project in the portfolio. It evolved from a basic RAG demo into a fuller research workflow platform.

Key points recruiters should understand:

- it uses a FastAPI backend
- it is designed around retrieval, analysis, and report generation
- it is structured for production-style growth rather than being a one-off notebook demo
- it powers the Ask direction for the portfolio by showing how portfolio content can connect to an LLM-backed retrieval flow

What it proves:

- backend architecture skill
- AI product thinking
- ability to move from prototype to platform shape
