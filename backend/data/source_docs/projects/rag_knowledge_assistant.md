# RAG Knowledge Assistant

RAG Knowledge Assistant is a deployment-ready retrieval-augmented generation project designed to prove end-to-end AI shipping discipline.

Recruiter highlights:

- FastAPI backend with a production-style API surface
- PostgreSQL and pgvector for retrieval storage
- Docker Compose and nginx for deployable infrastructure
- ingestion, retrieval, citations, and evaluation endpoints
- public VPS deployment, not just local notebook experimentation

What it proves:

- understanding of retrieval-augmented generation architecture
- backend and infrastructure ownership
- ability to move from concept to public deployment
